| Роль | Права роли | Группы пользователей |
| :--- | :--- | :--- |
| **cluster-admin** | Полный доступ ко всем ресурсам кластера (секреты, CRD, узлы, роли). | Инженеры инфраструктуры, DevOps-инженеры, руководитель IT, специалист ИБ |
| **security-auditor** | Просмотр всех ресурсов, включая секреты, логи аудита, RBAC-политики. Без права изменения. | Специалист ИБ, внутренний/внешний аудитор |
| **cluster-maintainer** | Управление узлами, namespaces, NetworkPolicy. Без доступа к секретам. | SRE, администраторы инфраструктуры |
| **devops-engineer** | Управление ресурсами в назначенных namespaces (деплои, сервисы, configmaps). Без секретов и узлов. | DevOps-инженеры (по отделам) |
| **developer-readonly** | Только просмотр подов, сервисов, логов в своих namespaces. Без exec и секретов. | Разработчики, тестировщики, аналитики |
| **ci-cd-robot** | Автоматическое развертывание в своих namespaces (создание/обновление ресурсов). Без удаления namespaces. | CI/CD-сервисы (GitLab CI, Jenkins) |
| **monitoring-robot** | Чтение метрик и конфигурации мониторинга. Без изменений и секретов. | Prometheus, Grafana, Loki/ELK |
